# TallerWDM-TEL317
Repositorio para desarrollar un análisis comparativo topológico de algoritmos RMLSA estático en Redes Ópticas Elásticas Estáticas.

Para este taller, se utilizará como base la investigación realizada en el artículo de _"Topology Bench: Systematic Graph Based Benchmarking for Optical Networks"_ (https://arxiv.org/pdf/2411.04160), haciendo uso de las directrices en su repositorio correspondiente: https://github.com/TopologyBench/topologybench.github.io
